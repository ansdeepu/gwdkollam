# **App Name**: GWD Dashboard 10082025

## Core Features:

- File Upload: Drag-and-drop file upload zone for easily adding files.
- File List Display: Display a list of uploaded files with their names.
- Clear File List: Button to clear all loaded files from the list.
- Error Handling: Provide user-friendly error messages.
- Intelligent Highlighting: Utilize a generative AI tool to determine if a selected file should be highlighted to the user, based on its content and recent user interactions.

## Style Guidelines:

- Primary color: Subtle blue (#8AB4F8) for a calm and focused feel.
- Background color: Light gray (#F5F5F5) for a clean and minimalistic aesthetic.
- Accent color: Slightly darker blue (#6096FF) for interactive elements.
- Body and headline font: 'Inter', a sans-serif, for a modern and objective look.
- Clean and minimalistic layout to ensure focus on the files.
- Subtle animations on file upload and deletion for enhanced UX.
